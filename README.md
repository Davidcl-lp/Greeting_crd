Primer proyecto Android con Jetpack Compose

Este proyecto es el resultado del **primer codelab del curso *Aspectos
básicos de Android con Compose*** de Google. En él aprendí a utilizar
Android Studio y crear mi primera aplicación sencilla usando Jetpack
Compose.

------------------------------------------------------------------------

## Trabajo realizado

### Creación del proyecto

-   Abrí **Android Studio** y generé un nuevo proyecto mediante\
    `New Project > Empty Activity`.
-   Configuré el nombre del proyecto y los parámetros iniciales.
-   Habilité el modo **Edge-to-Edge** y configuré el contenido usando
    `setContent`.

### Uso del Preview

-   Utilicé el panel **Preview** para ver cómo se renderizaba la
    interfaz sin tener que ejecutar la app en un dispositivo o emulador.

### Introducción a Jetpack Compose

-   Creé mi primera función **@Composable**, llamada `Greeting`.
-   Aprendí que:
    -   Las funciones componibles comienzan con mayúscula.
    -   Deben llevar la anotación `@Composable`.
    -   No devuelven vistas tradicionales: describen la UI de forma
        declarativa.
-   Usé un **Surface** y un **Text** para mostrar un mensaje.
-   Apliqué un **Modifier** para añadir padding y decorar elementos.

### Uso de Material Design 3

-   Apliqué colores y estilos usando la plantilla generada por Android
    Studio.
-   Utilicé el tema `GreetingCrdTheme` para envolver los componentes.
